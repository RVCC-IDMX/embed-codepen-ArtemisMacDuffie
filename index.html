Pen: Parallax Star background in CSS

by sarazond

This pen demonstrates the incredible versatility of CSS, as it uses only CSS to create an animated background that appears as stars of various sizes moving across the screen at different speeds, as if in parallax with the viewer. Features of CSS and HTML used are:

  • Functions
  • Psuedo-classes
  • Psuedo-elements
  • Keyframes
  • Animation properties
  • Element IDs
